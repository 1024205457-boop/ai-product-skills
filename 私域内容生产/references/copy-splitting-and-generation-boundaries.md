# Copy Splitting and Generation Boundaries

This reference shows how to turn one private-domain push copy into an implementable workflow. The key question is not "can AI write copy?" but "which words must be fixed, which can rotate, and which can be freely generated?"

## Example Source Copy

```text
小雨妈妈您好，今晚 20:00 有一节一年级数学思维直播课，主题是“图形找规律”。孩子上次测评显示图形题正确率偏低，建议先听这节课补一下。点击这里预约：{{link}}。如不想接收提醒，请回复 TD。
```

## Split the Copy

| Part | Example | Owner | How to implement |
| --- | --- | --- | --- |
| User field | 小雨妈妈 | Code | Read from CRM field `parent_nickname` |
| Time field | 今晚 20:00 | Code | Read from schedule table `lesson_start_at` |
| Grade/course | 一年级数学思维 | Code | Read from product/course table |
| Topic | 图形找规律 | Code | Read from lesson metadata |
| Learning signal | 图形题正确率偏低 | Code + rules | Read from assessment tag, never invent |
| Recommendation reason | 建议先听这节课补一下 | Template rotation | Choose from approved reason templates |
| CTA | 点击这里预约 | Template rotation | Choose from CTA pool by channel |
| Link | `{{link}}` | Code | Generate tracked URL |
| Opt-out | 回复 TD | Code/fixed policy | Always append if channel requires it |
| Tone rewrite | 更温和/更紧迫/更短 | AI | Rewrite only around approved facts |

## Code Insert Fields

These fields should never be generated by the model:

```json
{
  "parent_nickname": "小雨妈妈",
  "student_grade": "一年级",
  "course_name": "数学思维",
  "lesson_start_at": "今晚 20:00",
  "lesson_topic": "图形找规律",
  "weakness_tag": "图形题正确率偏低",
  "booking_link": "https://example.com/book?uid=1001",
  "opt_out_text": "如不想接收提醒，请回复 TD"
}
```

## Rotating Template Pool

Use template pools for controllable variation:

```json
{
  "opening": [
    "{{parent_nickname}}您好",
    "{{parent_nickname}}，提醒您一下",
    "{{parent_nickname}}，这节课和孩子最近的薄弱点比较相关"
  ],
  "reason": [
    "孩子上次测评显示{{weakness_tag}}，建议先听这节课补一下。",
    "这节课正好对应{{weakness_tag}}，可以作为一次针对性练习。",
    "如果最近想补{{lesson_topic}}，这节课比较适合先看。"
  ],
  "cta": [
    "点击这里预约：{{booking_link}}。",
    "可以先帮孩子占一个名额：{{booking_link}}。",
    "预约入口在这里：{{booking_link}}。"
  ]
}
```

## AI Free Generation Boundary

The model may generate:

- Shorter version for one-to-one chat.
- Warmer version for parent group.
- More action-oriented version for deadline reminder.
- 3 title variants for a poster/card.

The model must not generate:

- New course time.
- New discount or price.
- New learning diagnosis.
- Guaranteed outcome such as "一定提分".
- Fake scarcity such as "仅剩 3 个名额" unless sourced.

## Minimal Code Shape

```js
function renderTemplate(template, fields) {
  return template.replace(/\{\{(\w+)\}\}/g, (_, key) => fields[key] ?? "");
}

const fields = {
  parent_nickname: "小雨妈妈",
  lesson_start_at: "今晚 20:00",
  course_name: "一年级数学思维",
  lesson_topic: "图形找规律",
  weakness_tag: "图形题正确率偏低",
  booking_link: "https://example.com/book?uid=1001",
  opt_out_text: "如不想接收提醒，请回复 TD"
};

const template = "{{parent_nickname}}您好，{{lesson_start_at}}有一节{{course_name}}直播课，主题是“{{lesson_topic}}”。孩子上次测评显示{{weakness_tag}}，建议先听这节课补一下。点击这里预约：{{booking_link}}。{{opt_out_text}}";

console.log(renderTemplate(template, fields));
```

## Suggested AI Prompt

```text
你是私域运营文案助手。只能基于我给你的字段改写，不得新增时间、价格、承诺、名额、诊断结论。

字段：
- 家长称呼：小雨妈妈
- 课程时间：今晚 20:00
- 课程：一年级数学思维
- 主题：图形找规律
- 学习信号：图形题正确率偏低
- 链接：https://example.com/book?uid=1001
- 退订语：如不想接收提醒，请回复 TD

任务：
生成 3 条微信私聊提醒，每条 80 字以内。
每条都要包含时间、主题、推荐理由、链接、退订语。
语气自然，不要制造焦虑。
```
