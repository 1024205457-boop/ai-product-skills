# AI Demo Scorecard

| Factor | Strong signal |
| --- | --- |
| JD fit | Directly maps to target job keywords |
| User value | Solves a real user decision or task |
| AI depth | Uses retrieval, tools, memory, evaluation, or workflow |
| Product completeness | Has onboarding, output, error handling, and metrics |
| Demo feasibility | Can be built quickly with limited scope |
| Interview value | Produces concrete decisions to discuss |

## Recommended Demo Types

1. AI shopping/search guide.
2. Task-completion Agent with tools.
3. AIGC generation plus review loop.
4. B-end workflow automation assistant.
5. Data update and anomaly reporting copilot.
