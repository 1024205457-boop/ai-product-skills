# Query Taxonomy

| Type | User intent | Product response |
| --- | --- | --- |
| Navigational | Find a known item | Return exact item and shortcuts |
| Informational | Learn facts | Answer with citations |
| Comparative | Compare options | Structured comparison table |
| Constraint-based | Filter by needs | Ask only missing critical constraints |
| Transaction-oriented | Buy, book, apply, submit | Guide next action with risk checks |
| Troubleshooting | Diagnose issue | Step-by-step resolution |
| Ambiguous | Unclear goal | Make best effort, then ask one clarifying question |

## Shopping Guide Mapping

Map free-form user needs to:

- Need category
- Must-have constraints
- Nice-to-have preferences
- Risk exclusions
- Product candidates
- Evidence fields
- Recommendation reason
